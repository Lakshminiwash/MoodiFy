Moodify-mood based music app

Moodify is a full-stack MERN web application that plays songs according to the user's mood. The application uses facial expression
detection to identify emotions and provide a personalized music experience.

features :-
=> User Authentication (Register/Login)
=> JWT-based Authentication
=> Redis Token Blacklisting for Secure Logout
=> Mood Detection using Facial Expressions
=> Play Songs Based on User Mood
=> Responsive UI using React and SCSS
=> REST API Integration
=> Full-Stack Deployment on Render

🛠️ Tech Stack

Frontend:-
=> React.js
=> SCSS
=> Axios
=> React Router DOM

Backend:- 
=> Node.js
=> Express.js

Database:-
=> MongoDB
=> Redis

Other Tools & Libraries:-

=> JWT Authentication
=> Multer
=> ImageKit
